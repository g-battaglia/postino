# 3. La Dashboard Web

La Dashboard Web è il tuo centro di controllo principale in Postino. Vi accedi tramite il tuo browser web effettuando il login con il tuo account amministratore.

## Accesso (Login)
Naviga verso l'URL della tua installazione di Postino e inserisci nome utente e password.

## Navigare nella Dashboard
Una volta effettuato l'accesso, vedrai un'interfaccia pulita e veloce. Le sezioni principali a cui puoi accedere dal menu laterale sono:

- **Dashboard / Home:** Mostra una panoramica ad alto livello del tuo pubblico, della crescita recente e delle statistiche rapide.
- **Subscribers (Iscritti):** Dove puoi visualizzare, cercare e modificare le singole persone nella tua mailing list.
- **Campaigns (Campagne):** Dove scrivi e invii newsletter singole.
- **Sequences (Sequenze):** Dove imposti le email automatiche a goccia (come l'onboarding).
- **Templates:** Dove progetti visivamente l'aspetto delle tue email.
- **Settings (Impostazioni):** Dove configuri il tuo provider email e le preferenze GDPR.

L'interfaccia è progettata per essere intuitiva: basta cliccare sulla sezione che ti serve e seguire i pulsanti sullo schermo per creare o modificare gli elementi.
